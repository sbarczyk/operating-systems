# Systemy operacyjne

## Opis
Zbiór ćwiczeń laboratoryjnych z kursu **Systemów Operacyjnych** realizowanego na Akademia Górniczo‑Hutnicza w Krakowie (Wydział Informatyki).  
Materiały obejmują zagadnienia związane z zarządzaniem procesami, wątkami, synchronizacją, pamięcią oraz systemem plików w języku C.

## Technologie i umiejętności
Projekt pozwolił mi zdobyć praktyczne doświadczenie w:
- programowaniu w języku C w kontekście systemów operacyjnych,
- zarządzaniu procesami i wątkami oraz synchronizacji (mutexy, semafory, bariery),
- obsłudze pamięci i mechanizmów systemu plików,
- pracy z systemem kontroli wersji Git i organizacji kodu w repozytorium.

## Autorski wkład
Projekt realizowany w ramach zajęć akademickich — wszystkie zadania zostały opracowane przeze mnie z zachowaniem dobrych praktyk kodowania i dokumentacji.
